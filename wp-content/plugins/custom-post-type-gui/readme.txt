== Description ==

Create new custom post in easy and fast way.

Features:

* Create new custom posts as many as many as you want.

== Installation ==

* Login to WordPress admin panel
* Go to Plugins > Add New
* Click Upload 
* Click Choose File and select the Custom-Post.zip.
* Click Install Now.
* Activate the plugin through the 'Plugins' menu in WordPress.
* Go to Dashboard -> Wordpress Custom Post and create new custom posts.

OR

* Extract Custom-Post.zip
* Go to /wp-content/plugins/ and upload the extracted folder in this directory.
* Activate the plugin through the 'Plugins' menu in WordPress.
* Go to Dashboard -> Wordpress Custom Post and create new custom posts.
